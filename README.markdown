# Stations Website

## Dependencies

- [Jekyll][jekyll]
- [GitHub Pages][ghp]

## Getting Started

Install Jekyll:

    gem install bundler jekyll

Install other dependencies:

    bundle install

## Commands

### Update dependencies:

    bundle update

### Build:

    bundle exec jekyll build

The output will be located in `_site`

### Serve locally:

    bundle exec jekyll serve

A copy will be available at `http://localhost:4000`
